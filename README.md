House Buy and Sell System

A full-stack web application designed for property management, allowing users to buy, sell, or rent houses efficiently.
Key Features
- User Roles: Distinct dashboards for Admins, Property Sellers, and Buyers.
- Property Listings: Advanced search and filter options for finding houses by location, price, and size.
- Management System: Admin dashboard to manage property listings and user activity.
- Database Integration: Secure data handling using PHP and MySQL.

Technologies Used
- HTML5 & CSS3
- JavaScript
- PHP
- XAMPP (Local Server)
